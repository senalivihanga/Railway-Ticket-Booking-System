Railway Ticket Booking System

Overview

This is a web-based Railway Ticket Booking System developed using PHP and MySQL. The system allows passengers to search trains, book tickets, manage bookings, and enables administrators to manage train information.

Features

* User Registration and Login
* User Authentication
* Search Available Trains
* Book Railway Tickets
* View My Bookings
* Cancel Bookings
* Ticket Printing
* Payment Module
* Admin Dashboard
* Add, Update and Delete Train Details
* Booking Reports
* MySQL Database Integration

Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript
* XAMPP

Installation

1. Install XAMPP.
2. Copy the project folder into the `htdocs` directory.
3. Import the SQL database into phpMyAdmin.
4. Start Apache and MySQL.
5. Open the project in your web browser.
